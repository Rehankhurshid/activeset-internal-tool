# 5. Webflow Build Checklist

- [ ] `ALT Text` is configured for each Locale
- [ ] Set `HiDPI Dimension` for each image (Make sure Height is set to Auto)
- [ ] Disable Scroll when the Nav Menu is open ⚠️
- [ ] Use `Aspect Ratio` on Images on First Fold to reduce `CLS`
- [ ] Add `aria-label` to the link without descriptive text. (Ex. On LinkedIn Link with Icon, add "aria-label=Go to {Company} LinkedIn Profile")
- [ ] All the Image are `WebP`
- [ ] Meta Title + Description & Open Graph Image are configured
- [ ] Optimise the Site Performance
- [ ] Proper Semantic Structure
- [ ] Link Testing – Use a link crawler like the [W3C Link Checker](https://validator.w3.org/checklink), the Chrome plugin [Check My Links](https://chrome.google.com/webstore/detail/check-my-links/ojkcdipcgfaekbeaelaapakgnjflfglf?hl=en-GB), or [Screaming Frog](http://www.screamingfrog.co.uk/seo-spider/)
