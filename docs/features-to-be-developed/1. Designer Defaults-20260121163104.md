# 1. Designer Defaults

- [x] Clone [Relume Starter Template](https://webflow.com/website/relume-library-styleguide)
- [ ] Upload Fonts (Format: **WOFF2**)
- [ ]     Setup Styleguide:
    - [ ] Setup `body` tag style (font-family should be for Paragraph)
    - [ ]     Heading Tags
        - [ ] H1-H6 HTML Tags
        - [ ] `heading-style-h1` to `heading-style-h1-h6`
    - [ ]     Paragraph Tags
        - [ ] Style `All Paragraphs` (_if required_)
        - [ ] Style `Text Sizes` (ex. `.text-size-large`, `.text-size-medium` etc)
        - [ ] Create any additional styles if required
- [ ] Setup Color Variables ([Reference](https://finsweet.com/client-first/docs/variables))
