This page demonstrates different ways to embed the Project Links Widget.

‚ú® NEW: Script-Only Embedding (No DIV Required!)

The widget automatically injects itself at the script location.

0. Script-Only Embedding (Simplest)

No HTML div required - just add the script with data attributes

<!-- Widget automatically injects here -->
<script 
    src="https://your-domain.com/widget.js"
    data-auto-inject="true"
    data-project-id="your-project-id"
    data-theme="dark"
    data-show-modal="true"
></script>

0b. Script-Only with Static Links

Perfect for static websites with predefined links

<script 
    src="https://your-domain.com/widget.js"
    data-initial-links='[{"title":"Link 1","url":"https://example.com"}]'
    data-theme="dark"
></script>

1. JavaScript API (Programmatic)

Most flexible method with programmatic control

<div id="js-widget"></div>

<script src="https://your-domain.com/widget.js"></script>
<script>
    embedProjectLinksWidget('js-widget', {
        projectId: 'your-project-id',
        theme: 'dark',
        showModal: true
    });
</script>

2. Data Attributes (Declarative)

Declarative approach using HTML data attributes

<div 
    data-project-links-widget
    data-project-id="your-project-id"
    data-theme="dark"
    data-show-modal="true"
></div>

<script src="https://your-domain.com/widget.js"></script>

3. iFrame Embedding

Direct iframe for simple integration

<iframe 
    src="https://your-domain.com/embed?projectId=your-project-id&theme=dark"
    width="100%" 
    height="400" 
    frameborder="0"
    style="border-radius: 8px;"
></iframe>
