üìã Embed Code

To replace your current static dropdown, simply use this single script tag:

<!-- Replace your current static dropdown with this -->
<script 
    src="http://localhost:3000/widget.js"
    data-auto-inject="true"
    data-project-id="YOUR_PROJECT_ID"
    data-style="dropdown"
    data-position="bottom-right"
    data-show-on-domains="webflow.io">
</script>

üìù Configuration Options:

  • data-project-id: Get this from your dashboard URL or embed dialog
  • data-position: "bottom-right", "bottom-left", "top-right", "top-left"
  • data-show-on-domains: Comma-separated domains (e.g., "webflow.io,staging.com")
